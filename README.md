# 🚀 AI Job Board

A modern and responsive **Job Board Application** built using **React, Vite, Tailwind CSS, and React Router**. The application allows users to explore job opportunities, search and filter jobs, view detailed job descriptions, save jobs, and apply through an interactive application modal.

🔗 **GitHub Repository:**  
https://github.com/RekhaChittaloori/job-board-ai

> **Live Demo:** *(Add your Vercel URL here after deployment)*

---

## 📌 Features

### 🏠 Home Page
- Modern responsive landing page
- Hero section with call-to-action
- Working job title & location search
- Featured jobs section
- Top hiring companies with company logos
- Responsive navigation bar and footer

### 💼 Jobs Page
- Browse available jobs
- Search jobs by title
- Filter by location
- Filter by employment type
- Responsive job cards

### 📄 Job Details
- Complete job description
- Required skills
- Experience level
- Salary information
- Employment type
- Job ID
- Posted date
- Apply before date
- Responsibilities
- Requirements
- Benefits & perks

### 📥 Apply Feature
- Interactive Apply Now modal
- Resume upload
- Cover letter input
- Application success confirmation

### ❤️ Save Jobs
- Save/Unsave job functionality
- Dynamic button state

### 📱 Responsive Design
- Mobile-friendly layout
- Tablet support
- Desktop optimized

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/RekhaChittaloori/job-board-ai.git
```

Navigate into the project:

```bash
cd job-board-ai
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
